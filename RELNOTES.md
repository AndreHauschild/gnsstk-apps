GNSSTk-Apps 12.0.0 Release Notes
========================

 * This release introduces a new repo gnsstk-apps split from gnsstk repo.
 * It includes the following:
   * Toolkit name change from gpstk-apps to gnsstk-apps including a c++ namespace change to `gnsstk::`
   * Paradigm of separating apps versus libraries code into separate repos.
     * Gnsstk repo contains only libraries while the new gnsstk-apps repo contains only apps.  
 * Additionally, it contains minor bug fixes and a CI updates.

Updates since v11.2.1
---------------------

**Build System and Test Suite**
  * Remove python bindings for feature branch

**Gitlab CI**
  * Update removed conda py 27 builds
  * CI Updates for feature branch ci_issue_1 (TKS SPLIT OUT) big and small chains

**Library Changes**
  * Rename gpstk library/package/namespace to gnsstk.
  * split gpstk apps and library
  * Deprecate mergeRinObs application in favor of RinEdit which supports RINEX 3

Fixes since v11.0.0
--------------------
  * Fix name references in build.sh to use new name GNSSTk-Apps
